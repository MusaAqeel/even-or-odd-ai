# is-even-odd-ai

A simple NPM package that uses OpenAI's GPT-4o to determine if a number is odd or even.

## Installation 

```bash
npm install is-even-odd-ai
```


## Usage

```javascript
const { isOdd } = require('is-even-odd-ai');

console.log(isOdd(1)); // true
console.log(isOdd(2)); // false
```
